    Paw-Plan is my Software Engineering Capstone Project. This project was developed by me and 4 other group members. Within this application, you can schedule when your dog needs to be walked and fed and Paw-Plan will send a notification based on the selected times. The notifications will be categorized and it will specify if it is a walk reminder, a meal reminder, a vet appointment reminder, a park/play date reminder, and other potential categories to be decided later. You can also decide when you want the notifications to be sent.

    Paw-Plan

    Nevin's Fitness Forge is a comprehensive fitness tracking website developed using React, Firebase, and Firestore. It allows users to calculate their BMI and BMR, log and track their daily caloric intake, and monitor their weight progress over time with interactive graphs. The platform features user authentication, responsive design, and secure data storage, providing personalized insights and progress tracking to help users achieve their fitness goals.

    Nevin's Fitness Forge

    Nevin's Food Finder is a sleek web application designed to help users discover random dining spots nearby. Users can enter a type of cuisine and location, and the app will provide a random restaurant with details like name, address, rating, and a map with routing directions. Built with React, it uses Axios for API requests, the Yelp API for restaurant data, and the HERE API for geolocation and mapping. The minimalist design ensures an intuitive user experience, making it easy to find great food around you.
